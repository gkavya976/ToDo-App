# To-Do List Application

## Project Description

This is a simple Console-Based To-Do List Application developed using Python. The application allows users to manage daily tasks efficiently through a command-line interface.

## Features

* Add new tasks
* View all tasks
* Remove tasks
* Search tasks by keyword
* Clear all tasks
* Display total number of tasks
* Save tasks permanently using a text file

## Technologies Used

* Python
* VS Code
* Text File Handling (`open()`)

## Files Included

* `todo.py` - Main Python program
* `tasks.txt` - Stores tasks permanently
* `README.md` - Project documentation

## How to Run

1. Clone the repository:

   ```bash
   git clone <repository-link>
   ```

2. Navigate to the project folder:

   ```bash
   cd ToDoApp
   ```

3. Run the application:

   ```bash
   python todo.py
   ```
